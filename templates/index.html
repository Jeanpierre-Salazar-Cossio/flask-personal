Mis Proyectos

Sistema de Gestión de Ventas

Este sistema automatiza los procesos de ventas y gestiona el control de inventarios en tiempo real. Incluye generación de reportes detallados, análisis de ventas por periodos, y administración de usuarios, todo desarrollado en PHP con MySQL.

Plataforma de Registro Académico

Es una aplicación web que facilita el manejo del rendimiento académico de estudiantes. Entre sus características destacan el registro de notas, envío de notificaciones personalizadas y la generación de reportes, implementada usando Laravel con integración de Bootstrap.

Aplicación de Reservas de Eventos

Sistema web desarrollado en C# que permite gestionar reservas para eventos y realizar pagos en línea. Cuenta con notificaciones automatizadas, integración de métodos de pago, y un panel administrativo para los organizadores de eventos.
